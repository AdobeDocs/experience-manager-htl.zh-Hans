# HTL TCK {#the-htl-tck}

## HTL TCK 是什么

[HTML 模板语言](overview.md)以前称为 Sightly，随 [Adobe Experience Manager](http://www.adobe.com/cn/solutions/web-experience-management.html) 6.0 引入，作为适用于 HTML 的首选和推荐的服务器端模板系统取代了 JSP (Java Server Pages)。

HTML 模板语言技术兼容性工具包 (HTL TCK) 提供了一套测试，旨在证明 HTL 实施符合官方的[语言规范](https://github.com/adobe/htl-spec)。
