# HTL规范

HTML模板语言(HTL)曾被Adobe Experience Manager6.0引入，并取代JSP(JavaServer页面)作为HTML首选的服务器端模板系统。

这是定义语法和语言行为的HTL规范。

完整的HTL规范详见 [Open Source Git存储库](https://github.com/adobe/htl-spec)
