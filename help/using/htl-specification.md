# HTL规范

HTML模板语言（HTL，以前称为Sightly）已随Adobe Experience Manager 6.0引入，并取代JSP(JavaServer Pages)作为首选和推荐的HTML服务器端模板系统。

这是定义语言的语法和行为的HTL规范。

完整的HTL规范在我们的开放源 [Git存储库中详细介绍](https://github.com/adobe/htl-spec)
