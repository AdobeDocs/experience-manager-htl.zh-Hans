# 标准化 {#standardization}

以下是HTL标准：

* [HTL规范](https://github.com/Adobe-Marketing-Cloud/sightly-spec)\
   对于语言律师，正式语法和功能描述。

* [HTL TCK](https://github.com/Adobe-Marketing-Cloud/sightly-tck)\
   用于验证语言兼容性的测试套件。
