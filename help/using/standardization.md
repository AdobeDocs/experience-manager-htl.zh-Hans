# 标准化 {#standardization}

以下是 HTL 标准：

* [HTL 规范](https://github.com/adobe/htl-spec) - 对于语言专家而言，为正式的语法和功能描述。
* [HTL TCK](https://github.com/adobe/htl-tck) - 用于证明语言兼容性的一套测试。
